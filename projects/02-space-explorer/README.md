# Project 2: Space Explorer Video Generator!

## Hey Krwutarth! Let's Explore Space!

This project creates amazing videos about space and astronomy! Planets, stars, black holes, and more!

## What This Project Does

1. Writes a script about space facts using AI
2. Creates a voiceover using Edge TTS
3. Downloads space video clips from Pexels
4. Combines everything into a YouTube video
5. Creates a thumbnail
6. Generates title, description, and tags

## How to Run

1. Make sure Ollama is running: `ollama serve`
2. Navigate to this folder in Command Prompt
3. Run: `python run_project.py`
4. Find your video in the `output` folder!

## Example Topics You Can Try

Change the `TOPIC` in config.py to:
- "5 Mind-Blowing Facts About Black Holes"
- "What Would Happen If You Fell Into Jupiter"
- "The Biggest Stars in the Universe"
- "Is There Life on Mars?"
- "How the Moon Was Formed"
