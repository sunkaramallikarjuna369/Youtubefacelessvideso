# Project 3: Animal Kingdom Video Generator!

## Hey Krwutarth! Let's Make Animal Videos!

This project creates amazing videos about all kinds of animals!

## How to Run

1. Make sure Ollama is running: `ollama serve`
2. Run: `python run_project.py`
3. Find your video in the `output` folder!

## Example Topics

- "5 Cutest Baby Animals"
- "The Fastest Animals on Earth"
- "Animals with Superpowers"
- "Weirdest Animals You've Never Seen"
