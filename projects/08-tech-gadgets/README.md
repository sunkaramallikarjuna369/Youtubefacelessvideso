# Project 8: Tech & Gadgets Video Generator!

## Hey Krwutarth! Let's Make Tech Videos!

This project creates awesome technology fact videos!

## How to Run

1. Make sure Ollama is running: `ollama serve`
2. Run: `python run_project.py`
3. Find your video in the `output` folder!

## Example Topics

- "5 Coolest Gadgets of the Future"
- "How the Internet Actually Works"
- "Amazing Robot Facts"
- "The History of Video Games"
