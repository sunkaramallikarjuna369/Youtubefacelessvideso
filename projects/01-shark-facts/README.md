# Project 1: Shark Facts Video Generator!

## Hey Krwutarth! Let's Make Shark Videos!

This project creates amazing videos about sharks automatically! Just run the script and watch the magic happen!

## What This Project Does

1. Writes a script about shark facts using AI
2. Creates a voiceover using Edge TTS
3. Downloads shark video clips from Pexels
4. Combines everything into a YouTube video
5. Creates a thumbnail
6. Generates title, description, and tags

## How to Run

### Step 1: Make Sure Ollama is Running
Open Command Prompt and type:
```
ollama serve
```
Leave this window open!

### Step 2: Open Another Command Prompt
Navigate to this project folder:
```
cd C:\path\to\Youtubefacelessvideso\projects\01-shark-facts
```

### Step 3: Run the Project!
```
python run_project.py
```

### Step 4: Wait for Magic!
The script will:
- Write a script (30-60 seconds)
- Create voiceover (10 seconds)
- Download videos (1-2 minutes)
- Assemble video (2-5 minutes)
- Create thumbnail (instant)

### Step 5: Find Your Video!
Look in the `output` folder for your finished video!

## Customizing Your Video

Open `config.py` to change:
- `TOPIC` - What specific shark topic to cover
- `NUM_FACTS` - How many facts to include
- `VOICE` - Which voice to use
- `VIDEO_LENGTH` - Target video length

## Example Topics You Can Try

Change the `TOPIC` in config.py to:
- "5 Scariest Shark Species"
- "How Sharks Hunt Their Prey"
- "Baby Sharks - Amazing Facts"
- "The Biggest Sharks Ever"
- "Why Sharks Are Important"

## Files in This Project
