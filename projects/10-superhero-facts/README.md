# Project 10: Superhero Facts Video Generator!

## Hey Krwutarth! Let's Make Superhero Videos!

This project creates awesome superhero trivia videos!

## How to Run

1. Make sure Ollama is running: `ollama serve`
2. Run: `python run_project.py`
3. Find your video in the `output` folder!
