## Before You Start

Make sure you have:
1. Python installed (see 00-KRWUTARTH-START-HERE)
2. Ollama running (`ollama serve` in Command Prompt)
3. Your Pexels API key (free from pexels.com/api)

## How to Run Any Project

1. Open Command Prompt
2. Navigate to the project folder:
   ```
   cd C:\path\to\Youtubefacelessvideso\projects\01-shark-facts
   ```
3. Run the project:
   ```
   python run_project.py
   ```
4. Wait for your video to be created!
5. Find your video in the `output` folder!

## What Each Project Creates

Every project automatically creates:
- A video script (using AI)
- A voiceover (using Edge TTS)
- Downloads relevant video clips
- Assembles everything into a final video
- Creates a thumbnail
- Generates title, description, and tags

## Tips for Success

1. **Start with Project 1** - It's the easiest!
2. **Make sure Ollama is running** before you start
3. **Be patient** - Videos take a few minutes to create
4. **Check your output** - Always watch your video before uploading
5. **Have fun!** - You're creating real YouTube content!

## Customizing Projects

Each project has a `config.py` file where you can change:
- The specific topic
- Number of facts
- Voice to use
- Video length
