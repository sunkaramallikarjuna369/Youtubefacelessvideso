# Project 6: History Mysteries Video Generator!

## Hey Krwutarth! Let's Explore History!

This project creates fascinating history videos!

## How to Run

1. Make sure Ollama is running: `ollama serve`
2. Run: `python run_project.py`
3. Find your video in the `output` folder!
