# Project 5: Science Experiments Video Generator!

## Hey Krwutarth! Let's Make Science Videos!

This project creates cool science fact videos!

## How to Run

1. Make sure Ollama is running: `ollama serve`
2. Run: `python run_project.py`
3. Find your video in the `output` folder!

## Example Topics

- "5 Cool Science Experiments You Can Do at Home"
- "Why the Sky is Blue"
- "How Volcanoes Work"
- "Amazing Chemistry Facts"
