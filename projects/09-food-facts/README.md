# Project 9: Food Facts Video Generator!

## Hey Krwutarth! Let's Make Food Videos!

This project creates yummy food trivia videos!

## How to Run

1. Make sure Ollama is running: `ollama serve`
2. Run: `python run_project.py`
3. Find your video in the `output` folder!

## Example Topics

- "5 Weird Food Facts You Won't Believe"
- "How Pizza Was Invented"
- "The Most Expensive Foods in the World"
- "Foods That Were Invented by Accident"
