# Faceless YouTube Video Generation - Complete Learning System

A comprehensive, step-by-step learning system for creating faceless YouTube videos as a side hustle. This course covers everything from niche selection to monetization, with a focus on **100% FREE tools** and **automation scripts** to minimize manual work.

## What You'll Learn

- How to create engaging faceless YouTube videos without showing your face
- Free AI tools for script writing, voice generation, and video creation
- Complete automation pipeline from topic to uploaded video
- YouTube SEO and optimization strategies
- Multiple monetization methods beyond just AdSense

## Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/sunkaramallikarjuna369/Youtubefacelessvideso.git
cd Youtubefacelessvideso
```

### 2. Start with the Beginner Guide
Open [00-BEGINNER-START-HERE/README.md](./00-BEGINNER-START-HERE/README.md) and follow the setup instructions.

### 3. Install Requirements
```bash
pip install edge-tts moviepy Pillow requests python-dotenv
```

### 4. Run Your First Automated Video
```bash
cd scripts
python master_automation.py --topic "5 AI Tools for Productivity" --length 8
```
